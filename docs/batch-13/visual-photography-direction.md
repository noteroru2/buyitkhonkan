# Visual and Photography Direction

## Direction: “ตรวจจริง สะอาด โปร่งใส”

- Tone: documentary-commercial ที่เป็นธรรมชาติ ไม่ stock, ไม่ marketplace, ไม่ภาพ AI เสมือนจริง.
- Color temperature: neutral 4800–5200K; white balance สม่ำเสมอ. ใช้ navy/ink, white, light gray เป็นฐาน และสีส้มแบรนด์เป็น prop/accent ไม่เกิน 10% ของ frame.
- Light: soft directional, bright midtones, controlled screen reflection; ห้าม HDR/clarity จัดหรือ cyan-orange grade.
- Background: โต๊ะตรวจสีเทา/ขาวสะอาด, เครื่องมือจริง, ไม่มีโลโก้คู่แข่งเด่นหรือข้อมูลลูกค้า.
- Lens: 35–50mm equivalent สำหรับ process/environment, 70–85mm สำหรับ detail/portrait; หลีกเลี่ยง ultra-wide ที่ทำให้เครื่องบิด.
- Framing: medium/close detail, มือกำลังทำงานจริง, จัด product เป็นระเบียบ ไม่กองของแบบร้านของเก่า.
- Product: orientation และ scale คงที่; จอปิด/หน้าจอ demo ไม่มีบัญชี; serial ถูกปิดตั้งแต่ก่อนถ่าย.

## UI treatment

- Container radius 12–16 px, border `#e5e7eb`, shadow เบาเทียบ card ปัจจุบัน; ห้าม glow/neon.
- Caption 13–14 px สี ink-soft, ชิดซ้าย, ระบุ evidence/illustration ชัด.
- Overlay ใช้เฉพาะเมื่อ contrast ≥4.5:1; prefer separate HTML copy. Gradient overlay navy 55–75%, ไม่ใส่ข้อความใน bitmap.
- Mobile ใช้ crop เฉพาะ subject, สูงไม่เกิน 260–300 px above fold, object-position จาก focal metadata; อย่าโหลด desktop crop.
- Icon system: rounded 1.5–2 px stroke, optical 24/40 px, navy + single brand accent; emoji ควรถูกแทนแบบค่อยเป็นค่อยไปใน implementation.

ความแตกต่างจากคู่แข่งเกิดจาก evidence จริงและการอธิบายขั้นตอน ไม่เลียนแบบ Advice/JIB/Banana และไม่สร้างภาพหน้าร้านที่ไม่มีจริง.
